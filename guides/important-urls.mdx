Keep these URLs handy for your integration:

| Service | URL | Description |
| :--- | :--- | :--- |
| **Console** | `https://console.16tms.com` | Manage your account, games, and tournaments. |
| **Swagger UI** | `https://api.16tms.com/swagger` | Interactive API documentation. |
| **API Base URL** | `https://api.16tms.com/v1` | Base URL for all API requests. |
| **Bracket View** | `https://bracket.16tms.com` | Public view for tournament brackets. |
