# Games and Currencies

Before creating tournaments, you need to define the **Games** you support and the **Currencies** used for entry fees and prizes.

## Creating a Game

1.  Go to the **Games** section in the Console.
2.  Click **Add Game**.
3.  Fill in details: Name, Icon, Platform (Mobile/PC), etc.

## Creating Currencies

1.  Go to **Finance** > **Currencies**.
2.  Click **Add Currency**.
3.  Define the currency code (e.g., `USD`, `COIN`), symbol, and conversion rates if applicable.
