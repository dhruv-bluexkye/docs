# Creating Tournaments

To create a tournament:

1.  Navigate to **Tournaments** > **Create**.
2.  **General Info**: Name, Description, Game, Mode (1v1, 5v5).
3.  **Schedule**: Start date, Check-in time.
4.  **Format**: Select bracket type.
5.  **Prizes**: Define distribution.

Click **Publish** to make it live.
