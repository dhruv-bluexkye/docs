# Games

The **Games** section is where you define the titles your platform supports.

## Adding a Game

1.  Click **Add Game**.
2.  **Name**: Display name of the game (e.g., "Counter-Strike 2").
3.  **Key**: Unique identifier for API usage (e.g., `cs2`).
4.  **Modes**: Defined game modes (e.g., 5v5, 1v1).
5.  **Assets**: Upload box art and icons.

## Application
This metadata is used when creating tournaments to filter options and display correct info to end-users.
