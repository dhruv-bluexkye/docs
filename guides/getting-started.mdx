# Getting Started

Follow these steps to integrate 16TMS into your application.

## Prerequisites

Before you begin, ensure you have access to the **16TMS Console**.

## Quick Start Steps

1.  **Register** on the Console.
2.  **Generate API Keys**.
3.  **Create your Game** integration.
4.  **Set up Currencies**.
5.  **Create Tournaments**.

See the specific sections in the sidebar for detailed instructions on each step.
