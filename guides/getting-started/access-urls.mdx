# Access URLs

Here are the critical URLs you will need during development and operations.

## Interfaces

| Service | URL | Usage |
| :--- | :--- | :--- |
| **Console** | `https://console.16tms.com` | **Admin Dashboard**. Use this to create games, tournaments, and manage your organization. |
| **Swagger UI** | `https://api.16tms.com/swagger` | **API Playground**. Test endpoints interactively. |
| **Bracket Viewer** | `https://bracket.16tms.com` | **Public View**. A standalone view for tournament brackets (can be embedded). |

## API

| Environment | Base URL |
| :--- | :--- |
| **Production** | `https://api.16tms.com/v1` |
| **Staging** | `https://staging-api.16tms.com/v1` |
